### Spring的依赖注入   
IOC的作用：降低程序间的耦合度，依赖关系的管理都交给了Spring   
依赖关系的维护就称为依赖注入
####依赖注入：
>能注入的数据，有三类：
>>基本数据类型和String  
>>其他bean类型  
>>复杂类型/集合类型    
>>>复杂类型的注入：
>>>>具体可以看代码，但注意：
>>>>>用于给list结构结合注入的标签：   
>>>>>
>>>>>>list array set     
>
>>>>>用于给map结构结合注入的标签：  
>>>>>
>>>>>>map props  
>
>>>>>结构相同，标签可以互换
>
>注入的方式：       
>>使用构造函数提供     
>>使用set方法    
>>使用注解
